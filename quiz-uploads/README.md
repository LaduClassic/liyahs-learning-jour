# Quiz Upload Format

This folder contains quiz files for the Arabic section vocabulary practice.

## File Format

Each line in the quiz file should follow this format:

```
arabic|phonetic|definition
```

### Example (quiz1.txt):

```
شَعْرٌ|sha-r|hair
بُنِّيٌّ|bun-nii-y|brown
قَصِيرٌ|qa-sii-r|short
كَلْبٌ|kalb|dog
مُرَقَّطٌ|mu-raq-qat|spotted
تُمْسِكُ|tum-si-ku|she holds
لُعْبَةٌ|lu-bah|toy
كُرَةٌ|ku-rah|ball
الْمِضْرَبُ|al-mid-ra-bu|the racket
```

## Fields

- **arabic**: The Arabic word with proper diacritics (tashkeel)
- **phonetic**: Romanized pronunciation guide
- **definition**: English translation/meaning

## How to Use

1. In the Arabic section, click the Settings (gear) icon in Quiz Prep
2. Drag and drop your .txt file or click to browse
3. The words will automatically populate into all quiz modes:
   - Flashcards
   - Letter Order
   - Phonetic Match
   - Definition Match
   - Write It

## Tips

- Include diacritical marks (harakat) on Arabic words for proper pronunciation guidance
- Use simple, clear English definitions appropriate for the learning level
- Keep phonetic transcriptions consistent and easy to pronounce
- Test your quiz file after uploading to ensure all modes work correctly
